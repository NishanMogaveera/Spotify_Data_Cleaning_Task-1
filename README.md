# 🎵 Spotify Streaming History Data Cleaning

## 📌 Task Objective
The aim of this project is to clean raw Spotify streaming logs and convert them into an analysis-ready dataset.

## 🛠 Steps Performed
- Loaded dataset using pandas
- Removed duplicate rows
- Handled missing values
- Converted timestamp column to datetime
- Standardized text columns
- Created new feature: `hours_played`
- Exported cleaned dataset

## 📂 Files Included
- `spotify_history.csv` → Raw dataset
- `cleaned_spotify_history.csv` → Cleaned dataset
- `spotify_cleaning.ipynb` → Full preprocessing code

## 🚀 Tools Used
- Python
- Pandas
- NumPy
- Jupyter Notebook
